# Tests

Esta carpeta está preparada para contener las pruebas unitarias y de integración del proyecto.

## Estructura sugerida

```
tests/
├── test_models.py      # Pruebas de modelos de base de datos
├── test_routes.py      # Pruebas de endpoints/rutas
└── test_services.py    # Pruebas de lógica de negocio
```

## Ejecutar pruebas

```bash
pytest
```
